# Simple Calculator (Python)

This is a simple calculator made using Python.  
It can do basic math like adding, subtracting, multiplying, and dividing numbers.

I created this project to practice Python basics and understand how user input and conditions work.

---

## What this calculator can do

- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers

---

## How to run it

1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the file using this command:

   ```bash
   python calculator.py
